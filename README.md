NIFTY Quant Baseline Analysis

Overview- This repository contains a baseline quantitative analysis of the NIFTY 50 index using historical daily market data.
The project focuses on data cleaning, return analysis, strategy evaluation, and performance comparison against the market.

Contents:
nifty50_daily_project.ipynb- Jupyter notebook containing the complete analysis, strategy logic, backtesting, and performance metrics.
Nifty 50 Historical Data.csv- Raw historical daily OHLC data for NIFTY 50.
nifty50_clean_daily.csv- Cleaned and processed dataset used for analysis.
